# mvnrepository
General repository for maven

## Usage
pom.xml:
```xml
    <repositories>
        <repository>
            <id>hengyunabc-maven-repo</id>
            <url>https://raw.githubusercontent.com/clearriver/mvnrepository/master</url>
        </repository>
    </repositories>
or
    <repositories>
        <repository>
            <id>hengyunabc-maven-repo</id>
            <url>https://raw.github.com/clearriver/mvnrepository/master</url>
        </repository>
    </repositories>
```
